# 小白人工智能学习总览

## 适合对象
本知识库面向没有系统学习过人工智能的高中生、大学低年级学生、转专业学习者和零基础自学者。学习者不需要一开始掌握高等数学或复杂编程经验，但需要具备基本逻辑思维、持续练习能力和动手做小项目的态度。

## 人工智能是什么
人工智能不是让机器真的拥有人类意识，而是让计算机通过数据、规则、模型和算法完成过去通常需要人类智能参与的任务，例如识别图片、理解文本、进行推荐、完成分类、规划路径和辅助决策。

## 学习主线
人工智能入门建议按照“概念理解 → Python 基础 → 数据处理 → 机器学习 → 深度学习 → 大模型与应用 → 项目实践”的顺序推进。

## 不能跳过的能力
1. 会描述问题：知道要解决什么任务。
2. 会整理数据：知道数据从哪里来、是否干净、标签是否可靠。
3. 会训练模型：理解训练集、验证集、测试集。
4. 会评估结果：会看准确率、召回率、F1 等指标。
5. 会做项目：能够把一个想法变成可运行的小系统。

## 常见误区
只看视频不写代码，很难真正掌握；只追大模型热点，不理解基础机器学习，会导致知识断层；只关注模型效果，不关注数据质量和评估方法，容易得出错误结论。
