# 大模型基础

## 大模型是什么
大模型是在海量文本、代码或多模态数据上训练得到的模型，通常具有较强的语言理解、生成和推理能力。常见能力包括问答、总结、翻译、代码生成、信息抽取和对话。

## Prompt
Prompt 是用户给模型的输入指令。好的 Prompt 会说明任务目标、输出格式、约束条件和必要背景。

## 上下文窗口
大模型一次能处理的文本长度有限，这个限制称为上下文窗口。超出窗口的历史内容可能无法被模型看到。

## 幻觉
幻觉指模型生成看似合理但不真实或无依据的内容。减少幻觉的方法包括 RAG 检索、引用来源、工具调用、限制回答范围和人工审核。

## RAG 与微调的区别
RAG 更适合补充事实知识，知识更新方便；微调更适合学习回答风格、任务流程和稳定格式。
