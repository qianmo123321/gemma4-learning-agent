# 人工智能四周入门计划

## 第 1 周：Python 与计算思维
目标是掌握变量、条件、循环、函数和基本数据结构。练习任务包括猜数字游戏、成绩统计和文本词频统计。

## 第 2 周：数据处理与机器学习
目标是理解数据集、标签、训练集、验证集、测试集。学习线性回归、逻辑回归、KNN 和决策树。完成一个表格分类小项目。

## 第 3 周：深度学习与大模型
目标是理解神经网络、损失函数、训练过程、Transformer 和大模型基本概念。可以调用大模型 API 或本地模型完成问答任务。

## 第 4 周：RAG 与学习智能体项目
目标是完成一个小型 RAG 学习助手。功能包括上传资料、检索相关内容、回答问题、显示来源、保存对话和收集反馈。
