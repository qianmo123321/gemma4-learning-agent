# LoRA 微调入门

## LoRA 是什么
LoRA 是 Low-Rank Adaptation 的缩写，是一种参数高效微调方法。它不直接更新大模型全部参数，而是在部分层中加入低秩矩阵，只训练少量新增参数。

## 为什么使用 LoRA
完整微调大模型需要大量显存和训练数据。LoRA 能显著降低显存需求和训练成本，适合个人项目、小团队和教学实验。

## LoRA 适合学习什么
LoRA 适合让模型学习稳定行为模式，例如回答风格、固定输出格式、学习路径规划流程、出题和解析格式、错题讲解方式和项目问答规范。

## LoRA 不适合什么
LoRA 不适合频繁更新事实知识。如果只是补充课程资料、制度文件或项目文档，优先用 RAG。

## 数据质量
微调数据质量比数量更重要。低质量、错误或风格混乱的数据会让模型学坏。
