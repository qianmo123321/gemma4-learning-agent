# Python 入门路线

## 为什么先学 Python
Python 语法简洁，生态丰富，是人工智能入门最常用的语言。学习人工智能不要求一开始成为 Python 专家，但需要掌握足够完成数据处理、模型调用和简单项目开发的基础能力。

## 必学语法
变量和数据类型、列表、字典、条件判断、循环、函数、文件读写、异常处理和模块导入。

## 学习建议
学习 Python 不要只背语法，要用小任务练习。例如统计一段文本中每个词出现次数、读取 CSV 文件并计算平均分、写猜数字小游戏、写学生成绩管理小程序、调用一个简单 API 并保存结果。

## 与 AI 的关系
Python 是人工智能项目的胶水语言。数据清洗、模型训练、模型推理、Web 后端、实验记录等环节都可以用 Python 完成。
