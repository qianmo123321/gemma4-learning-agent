# LoRA/QLoRA 微调说明

LoRA 是一种参数高效微调方法。它不直接更新大模型全部参数，而是在注意力层等关键模块中插入低秩矩阵，只训练少量新增参数。这样可以显著降低显存占用和训练成本。

## LoRA 适合做什么

LoRA 适合让模型学习固定格式、固定风格、固定任务流程。例如学习路径规划、自动出题、错题讲解、项目文档问答风格、特定领域术语解释、标准化输出格式。

## LoRA 不适合做什么

如果只是补充事实知识，优先使用 RAG。因为知识更新频繁时，重新微调模型成本较高。LoRA 更适合学习行为模式，RAG 更适合补充外部知识。

## 推荐训练参数

对于 16GB 显存环境，建议：load_in_4bit = true，batch_size = 1，gradient_accumulation_steps = 8，max_length = 512 或 1024，lora_r = 8，lora_alpha = 16，epochs = 1 到 3。

## 训练结果判断

如果 train loss 下降但 eval loss 上升，可能过拟合。如果 loss 长期不下降，可能是数据质量差、学习率不合适或输出格式不统一。
