# 学生智能体功能设计

本项目面向学生学习场景，不只提供普通问答，还提供多个学习智能体。

## AI 知识问答 Agent

根据学生问题和本地知识库内容生成回答，并展示参考来源。

## 学习路径规划 Agent

根据学生的学习目标、基础水平、学习周期和每天可用时间，生成阶段化学习计划。输出通常包括阶段目标、每日任务、推荐资料、检测方式、复习安排。

## AI 出题 Agent

根据知识点自动生成题目，支持选择题、判断题、简答题和案例分析题。每道题需要包含答案和解析。

## AI 陪练聊天室 Agent

以老师、助教、面试官或同学身份进行多轮对话。它可以追问、提示、纠错和总结。

## 错题讲解 Agent

根据学生答错的题目，分析错误原因，给出正确思路，并生成类似题用于巩固。
