# 人工智能入门常见问题

## 没有数学基础能不能学 AI
可以入门，但后续想深入需要逐步补数学。入门阶段先理解概念和项目流程，再补线性代数、概率统计和微积分。

## 先学 Python 还是先学机器学习
建议先学 Python 基础，再学机器学习。因为机器学习实验需要用 Python 处理数据、调用库和分析结果。

## RAG 和 LoRA 先学哪个
如果目标是做私有知识问答，先学 RAG。如果目标是让模型形成固定回答风格，再学 LoRA。

## 初学者最适合做什么项目
推荐做小型项目，例如文本分类、电影评论情感分析、学生成绩预测、课程资料问答助手和错题讲解机器人。

## 学习时如何避免半途而废
不要只看课程，要每周完成一个小作品。项目越小越好，关键是跑通完整流程。
